# vim-ide-config

# install nerd font

You need to install nerdfont to render all glyphs properly.
on mac, use brew to install font
```shell
brew tap homebrew/cask-fonts
brew install --cask font-hack-nerd-font
brew install --cask font-hack-nerd-font
```
